Git commit:O que faz o comando git commit?
Este comando é como definir um ponto de verificação no processo de desenvolvimento
Git push:git push é mais usado para publicar modificações locais a um repositório central.
Git status:Verifica quais os estados que os arquivos estão. Se existe algum arquivo modificado, adiciona ou removido. 
Git rm :não removerá um arquivo apenas do seu diretório de trabalho.
Para criar um novo repositório, você vai usar o comando Git init.
